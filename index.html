Text to teapot
Text to teapot is a web journal about my personal experience in experimenting with GPT-3 and 3D modelling and the relationship I developed with it.

The thesis project aimed to test the current capabilities of Ai, its limitations and its implementation or replacement to the designer's work. While remaining an intimate production, the journal's hyperlinks and bibliography allow the reader to explore the technical stages of my research.

text-to-teapot.com (currently not mobile optimized)
Handyhands
Handy Hands is a project by Claudia Polizzi and Stefano Riba. I got to make the first version of this platform for the startup launch.

I developed the website according to the authors' graphical and functional guidelines and requests, using pure code front-end only.
Lottozero.org/directory
Implementation of the directory page for the Lottozero website, based on Claudia Polizzi's idea and design.

The page was implemented on the pre-existing Squarespace site, using a mix of the platform's features, plugins and pure code.
UI
Series of UI commissions, for subsequent development of web platforms and apps. Below are concepts for a website of a fair promoted by the Eurac research institute, a redesigned web app for safety inspections in companies, for use mainly tablet, and the platform of a craft fair.
BESSI'Studio
Showcase site for BESSI'Studio, redesigned from scratch and developed in pure code front end only.

The goal was to develop a platform for the studio with which to approach new and established clients in a more modern way, but still true to its identity.
